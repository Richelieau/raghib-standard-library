# JavaScript Standard Library Demo

Repository ini berisi contoh penggunaan **JavaScript Standard Library** dalam bentuk file HTML terpisah untuk setiap library bawaan JavaScript.  
Setiap file berisi:

- Contoh kode
- Output di console
- Penjelasan singkat fungsi library

Tujuan project ini adalah untuk memahami fitur standar JavaScript tanpa menggunakan library eksternal.

---

## üìö Daftar Standard Library yang Dibahas

### 1. Array

File: `array.html`  
Digunakan untuk menyimpan banyak data dalam satu variabel.  
Contoh method: map, filter, sort, length.

---

### 2. BigInt

File: `bigint.html`  
Untuk menangani angka yang sangat besar di luar batas Number biasa.  
Ditulis dengan akhiran `n`.

---

### 3. Boolean

File: `boolean.html`  
Tipe data logika: true / false.  
Digunakan dalam percabangan dan validasi.

---

### 4. Date

File: `date.html`  
Untuk mengelola tanggal dan waktu.  
Contoh: getFullYear(), getMonth(), getDate().

---

### 5. Encode

File: `encode.html`  
encodeURIComponent digunakan untuk membuat teks aman dimasukkan ke URL.

---

### 6. Eval

File: `eval.html`  
Menjalankan string sebagai kode JavaScript.  
‚ö† Tidak disarankan untuk penggunaan umum karena risiko keamanan.

---

### 7. JSON

File: `json.html`  
Untuk konversi object ‚Üî JSON string.  
Method utama:

- JSON.stringify()
- JSON.parse()

---

### 8. Map

File: `map.html`  
Struktur data key-value modern.  
Lebih fleksibel dari object biasa.

---

### 9. Math

File: `math.html`  
Berisi fungsi matematika bawaan:

- sqrt
- random
- pow
- PI

---

### 10. Number

File: `number.html`  
Fungsi tambahan untuk tipe angka.  
Contoh: toFixed(), isInteger().

---

### 11. Object

File: `object.html`  
Struktur dasar data JavaScript berbentuk pasangan key-value.  
Contoh: Object.keys(), Object.values().

---

### 12. Proxy

File: `proxy.html`  
Digunakan untuk memonitor dan mengontrol akses ke object.

---

### 13. Reflect

File: `reflect.html`  
API bantu untuk operasi object secara terstruktur.

---

### 14. RegExp

File: `regexp.html`  
Untuk pencarian dan pencocokan pola teks menggunakan regular expression.

---

### 15. sample.json

Contoh file data JSON untuk simulasi pertukaran data.

---

### 16. Set

File: `set.html`  
Kumpulan nilai unik (tidak boleh duplikat).

---

### 17. String

File: `string.html`  
Operasi manipulasi teks:

- toUpperCase
- includes
- slice

---

### 18. Symbol

File: `symbol.html`  
Tipe data untuk membuat key object yang unik.
